📆 구글 캘린더 연동
최초 1회 설정 방법 (구글 클라우드 · 3분)
  1. 클라우드 콘솔에서 프로젝트 생성
  2. Google Calendar API 사용 설정
  3. OAuth 동의 화면 → 외부 → 테스트 사용자에 내 Gmail 추가
  4. 사용자 인증 정보 → OAuth 클라이언트 ID → 앱 유형 데스크톱 앱 → 생성
  5. 나온 클라이언트 ID·비밀번호를 아래에 붙여넣기 → 저장 → 연결
※ 연결은 이 PC 브라우저(localhost)에서 진행하세요. 폰·외부주소에선 구글이 리디렉션을 막습니다.
